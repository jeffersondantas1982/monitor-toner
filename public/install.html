Guia de Instalação

Configurando o seu servidor de monitoramento

Voltar ao Manual

Pré-requisitos

Antes de começar, certifique-se de ter as seguintes ferramentas instaladas em seu servidor ou computador:

  • Node.js (LTS) O motor que executa o sistema. Recomenda-se a versão 18 ou superior. Baixar aqui
  • Git Necessário para clonar e atualizar o repositório. Baixar aqui

Passo a Passo de Instalação

  • Clonar o Repositório Abra o terminal (PowerShell ou Prompt de Comando) e execute:
    git clone https://github.com/jeffersondantas1982/monitor-toner.git
  • Acessar a Pasta Entre no diretório do projeto:
    cd monitor-toner
  • Instalar Dependências Baixe todos os pacotes necessários via NPM:
    npm install
  • Iniciar o Sistema Inicie o servidor de monitoramento:
    npm start

Acessando o Sistema

Uma vez iniciado, o painel estará disponível nos seguintes endereços:

Local: http://localhost:3000
Rede: http://[IP-DO-SERVIDOR]:3000

Para descobrir o IP do servidor, você pode usar o comando ipconfig no Windows ou ifconfig no Linux.

Script de Lançamento (Windows)

O projeto inclui um arquivo chamado subir_projeto.bat que pode ser adaptado para automatizar tarefas de atualização e inicialização.
